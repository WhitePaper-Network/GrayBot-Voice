# GrayBot-Voice

**GrayBot-Voice**は、全く新しいTTS Botです。 このボットは、声の種類が豊富なVoiceText Web APIを使っています。

English version is [here](https://github.com/WhitePaper-Network/GrayBot-Voice/blob/main/README-en.md)

## セットアップ

1. GitHubからダウンロード
2. npmでモジュールを入れる
3. config.jsonを編集する
4. GRAY_VOICE.jsを実行する
